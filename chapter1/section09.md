# 1.9 标注问题

$$T = \{ (x_1, y_1), (x_2, y_2), \dots, (x_N, y_N)\}$$ 

$$P (y^{(1)}, y^{(2)}, \dots, y^{(n)} | x^{(1)}, x^{(2)}, \dots, x^{(n)}\}$$ 

对 $$x_{N+1} = (x_{N+1}^{(1)}, x_{N+1}^{(2)}, \dots, x_{N+1}^{n})^T$$ 找到条件概率 $$P( (y_{N+1}^{(1)}, y_{N+1}^{(2)}, \dots, y_{N+1}^{(n)})^T | X)$$ 最大的标记序列 $$y$$ 。

常用的统计学习方法有隐马尔可夫模型和条件随机场。
